What is Squelch?

Squelch is the control that stops a radio from constantly playing background noise when there is no real signal to receive.

Simple Explanation

When a radio is turned on and no one is transmitting, the receiver still picks up random noise. Without squelch, you would hear that hiss all the time.

Squelch acts like a gate. If the incoming signal is too weak or does not meet the required condition, the speaker stays muted. When a valid signal comes in, the gate opens and you hear the transmission.

Why It Matters

Squelch makes radios usable. Without it, the speaker would often be full of constant static instead of only opening when there is something worth hearing.

How It Works

  • The radio listens for incoming signal energy.
  • If the signal is below the squelch threshold, audio stays muted.
  • If the signal is strong enough, the radio opens the speaker.
  • You then hear the voice, tone, or data transmission.

Types of Squelch

Carrier Squelch

This is the basic version. The radio opens whenever it detects a strong enough carrier signal. It does not care who sent it, only that a signal is there.

Tone Squelch (CTCSS)

This version only opens if the incoming transmission includes the correct sub-audible tone. That helps reduce unwanted traffic from other users on the same frequency.

Digital Coded Squelch (DCS)

This works similarly to tone squelch, but uses a digital code instead of a continuous tone. It serves the same basic purpose: keeping the speaker closed unless the right code is present.

What Happens if Squelch is Set Wrong?

  • Too low: the radio may open on noise and hiss constantly.
  • Too high: weak but real transmissions may never break through.
  • Wrong tone/code: the signal may be present, but you still hear nothing.

In Practice

If you turn the squelch control down too far, the radio becomes noisy. If you turn it up too far, you can miss weak signals. Good squelch setting is a balance between blocking unwanted noise and still hearing valid transmissions.

Quick Summary

Squelch is a filter that keeps your radio silent until a valid signal appears. Carrier squelch listens for signal strength, while CTCSS and DCS add extra conditions before the speaker opens.
